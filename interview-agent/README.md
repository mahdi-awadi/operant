# Interview Agent — راهنمای استفاده

این فولدر یک **دستیار مصاحبه** است که برای استخدام **کارآموز** طراحی شده.

## نحوه‌ی استفاده

۱. در روبیکا یا تلگرام، یک Claude session جدید spawn کنید با مسیر این فولدر:

```
/spawn interview /home/channelhub/interview-agent
```

۲. Claude به‌طور خودکار `CLAUDE.md` را می‌خواند و نقش دستیار مصاحبه را می‌گیرد.

۳. به محض شروع، از شما اسم کاندیدا را می‌پرسد.

۴. سپس ۱۸ سوال را **یکی‌یکی** می‌فرستد. شما جواب کاندیدا را در تلگرام تایپ کنید.

۵. در پایان، فایل کامل مصاحبه را در `candidates/<اسم-کاندیدا>-YYYY-MM-DD.md` ذخیره می‌کند و یک ارزیابی کوتاه می‌فرستد.

## دستورات حین مصاحبه

| دستور | کاربرد |
|--------|--------|
| (متن آزاد) | جواب کاندیدا برای سوال فعلی |
| `بعدی` | رد کردن سوال فعلی |
| `قبلی` | برگشت یک سوال |
| `چندتا مونده` | نمایش پیشرفت |
| `تمام` | پایان زودهنگام و ذخیره |

## ساختار فولدر

```
interview-agent/
  ├── CLAUDE.md         # دستورالعمل برای Claude (دستیار مصاحبه)
  ├── questions.md      # لیست ۱۸ سوال
  ├── README.md         # این فایل
  └── candidates/       # فایل‌های ذخیره‌شده‌ی مصاحبه‌ها
```

## بعد از مصاحبه

- فایل کاندیدا را در `candidates/` بازبینی کنید
- اگر کاندیدا قابل قبول است، تست‌های عملی (بخش ۴.۱ تا ۴.۶ راهنمای استخدام) را برگزار کنید
- راهنمای کامل استخدام: `/home/channelhub/docs/hiring-guide-fa.md`

## ویرایش سوالات

اگر می‌خواهید سوالات را تغییر دهید، فقط `questions.md` را ویرایش کنید. `CLAUDE.md` نیازی به تغییر ندارد.

## یادداشت فنی

- این agent از `reply` tool هاب استفاده می‌کند، نه stdout
- پاسخ‌ها در حافظه‌ی session نگه داشته می‌شوند و در پایان به فایل نوشته می‌شوند
- در صورت قطع session، پاسخ‌های ثبت‌شده تا قبل از قطعی از دست می‌روند — توصیه: مصاحبه را در یک جلسه‌ی پیوسته انجام دهید
